# Router 1×3 RTL Design

A Verilog HDL implementation of a 1×3 Packet Router with modular RTL design and functional verification.

## Modules
- FIFO
- Synchronizer
- Finite State Machine (FSM)
- Register Block
- Top Module

## Features
- Packet routing based on destination address
- FIFO-based buffering
- FSM-controlled data flow
- Functional verification using Verilog testbenches

## Tools
- Verilog HDL
- ModelSim
